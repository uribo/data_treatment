# Excelの正しい使い方と間違った使い方

「**ネ申Excel**」はやめましょう

## 正しい使い方

- [ ] 記録・計測したデータを各セルに入力する
- [ ] 印刷する、以外の用途を考える（Excelで文書を作ってはいけない）

## 間違った使い方

間違った使い方というのは不適切かもしれませんが、少なくとも、データ解析においてはやめてほしい、というもの。害悪度を５段階評価しています（★が多いほどやめてほしい）

- [ ] 余分な罫線 `★★☆☆☆`
- [ ] セルの結合 `★★★★☆`
- [ ] グラフの挿入 `★★★★★`
- [ ] セル内での改行 `★★★☆☆`
